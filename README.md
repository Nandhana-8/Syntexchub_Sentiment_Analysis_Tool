# Syntexchub_Sentiment_Analysis_Tool

## 📌 Overview
Sentiment Analysis Tool for AI Internship Project.Predicts whether user text is positive or Negative Using Machine Learning with CountVectorizer and Naive Bayes classifier

## 🚀 Features
✅ Text Preprocessing  
✅ CountVectorizer Feature Extraction  
✅ Naive Bayes Classifier  
✅ Accuracy Score  
✅ F1 Score  
✅ CLI User Input System  

## 🛠 Technologies Used
- Python
- Scikit-learn
- Machine Learning
- Natural Language Processing

## ▶️ How to Run in VS Code

1. Open Visual Studio Code  
2. Open the project folder  
3. Open terminal in VS Code  
4. Install required library:
pip install scikit-learn
5. Run the project:
python sentiment_analysis.py

Enter your review: This product is amazing  
Predicted Sentiment: Positive
